# Codex — Documentation Hygiene (Later)

## Role

Optional assistant for **formatting**, **linking**, and **bulk doc consistency**—not gameplay.

## What Codex should do

- Normalize headings, cross-links between specs and build log, and template adherence.
- Generate tables of contents or index files **only when requested**.
- Align terminology across `Docs/` and `agent_instructions/` without changing meaning.

## What to avoid

- Changing puzzle or lore meaning while “cleaning” copy.
- Unity asset or C# edits unless explicitly scoped.
- Introducing new systems or roadmap promises not agreed in `Docs/DECISIONS.md`.
